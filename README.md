# EduManage Pro — Primary School Management System

A full-stack web application for managing primary school operations, built with PHP and MySQL. It provides separate interfaces for administrators and students, covering the full school workflow from admission applications through to academic performance tracking.

## Features

### Public
- Responsive landing page with school highlights, about section, and call-to-action
- Online admission application form with document uploads (birth certificate, proof of address)
- Student login portal (name + class authentication)

### Student Dashboard
- Personalized welcome banner with profile picture upload
- View subject marks with animated progress bars and color-coded performance indicators
- Automatic grade calculation (A–F) per subject and overall average
- Download academic performance report

### Admin Panel
- Sidebar dashboard with live stats (total students, applications, marks entered)
- View and manage all submitted admission applications
- Register new students into the system
- Enter and record subject marks per student
- Recent applications table with payment status badges
